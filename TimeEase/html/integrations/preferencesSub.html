Preferences


Time Ease offers a few options that can be specified by the user.


1. Auto Login.
2. Work Offline.
3. Include already Billed Entries.
4. Default Client.
5. Default Project.
6. Default Task.
7. Help Contents options.
7. Time Ease Update Schedule.
8. Hot Keys for Time Ease actions.
9. Security options for encrypting FreshBooks Token and URL.


You can find these options by going to File --> Preferences. By default you must login and specify your application User Name,
FreshBooks Token, and Site URL. These will be saved for you, also you can specify at the Login dialog the Auto Login preference.
