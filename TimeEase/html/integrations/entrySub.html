Adding Time Entries

I have attempted to make adding time as easy as possible. Time Ease supports being offline and we will store any entry that gets created or updated during time offline. You will need to select Sync when back online to Sync the Time Ease state with FreshBooks. This Sync operation will happen when you first start up the application if you are online, otherwise waits for a manual Sync while online. Here are the Time Entry options:

1. Tracking Time.
   a. Select 'Track Time' --> 'Track Time', or select this icon from the Tool Bar: .
   b. This will open a new editor.
   c. The application will pull your default Clients, Projects, and Tasks; if you have set them up.
       i. If there isn't a default setup you will need to select a Client.
       ii. Now that you have selected the Client the Projects will now show, select one.
       iii. Again your Task selections will update when the Project is selected, please select a Task.
   g. Press Start to begin the timer, Stop to pause the time, or Reset to clear all selections and the timer.
   h. Press Send/Store when you are ready to send the transaction to FreshBooks.
   i. You must have a Client, Project, and Task selected to send your entry.


2. Clocking In and Out.
   a. Select this icon from the Tool Bar: , or select the Go-Time icon from the System Tray, right click and select 'Clock In'.
   b. This will register the start time.
   c. When ready to Clock Out select this icon from the .
       i. A pop up will display to allow you to enter the entry details. Your Time Ease settings will be selected by default.
       ii. If there isn't a default setup you will need to select a Client.
       iii. Now that you have selected the Client the Projects will now show, select one.
       iv. Again your Task selections will update when the Project is selected, please select a Task.
       iv. You are required to have all options set, except for the Notes as they are optional.
   d. Press 'Ok' when you are ready to send the transaction to FreshBooks, or 'Cancel' to not send the entry.


3. Adding Manual Entry.
   a. Select 'Track Time' --> 'Track Time', or select this icon from the Tool Bar: .
   b. This will open a new editor.
   c. You will be able to just add some time manually to the timer, defaults to '00:00:00'.
       i. The format is Hours:Minutes:Seconds, hh:mm:ss; ex. 01:32:01 is one hour, thirty two minutes and.
       ii. You are also able to just add 01 for a single hour, or 01:30 for an hour and a half.
       iii. Set the other entry details, add notes if you would like.
   d. Select Send/Store when you are ready to send to FreshBooks.
