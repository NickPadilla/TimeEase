Time Ease was developed for widespread use, and is in the early stages of feature implementations. I will be adding new features often, so check for upcoming updates. Currently we offer integration with FreshBooks, however working on the ability to use Time Ease as a stand alone Time Tracker is in the works. Currently we support these features:

1. Track time on FreshBooks Clients, Projects, and Tasks.
2. Manual time entry creation.
3. Update entries already put into FreshBooks, currently shows 7 days worth of working time.
4. Clock In/Out support for those who don't need a counter. Simply Clock In, then Out.
5. Offline support, after initial FreshBooks Sync.
6. Single instance supports multiple users.
7. Define system preferences and defaults, such as Default Client, Project, and Tasks.
8. Work completely offline, after initial FreshBooks Sync.
9. FreshBooks Sync, ensure your always up to date.
10. Report a bug or request a new feature from the Time Ease user interface.
11. Support for automatic updates, or manage update yourself by simply clicking 'Check for Updates' whenever your ready.
12. Contains Help documentation, as you can see, to help answer your Time Ease questions.
13. System Tray support, perform some functions without the application window open.


To enable Automatic Updates
1. Go to 'Time Ease' and select 'Preferences'.
2. Expand 'Install/Update' --> then select 'Automatic Updates'.
3. Select 'Automatically find new updates and notify me.'
4. Now select 'OK'.
5. Now every time Time Ease starts it will look for updates, it will allow you to say if to install it or not.
